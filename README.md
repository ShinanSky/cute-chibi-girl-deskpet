# Cute-chibi-girl-deskpet

<p align="center">
    <img src="docs/idle.png" width="220">
</p>

> A tiny AI-powered chibi girl who quietly lives on your desktop.

She walks around.

She falls asleep when she's tired.

She wakes up on her own.

She occasionally talks to you.

And whenever you want, she's always there to chat.

Cute Chibi Girl DeskPet is an open-source AI desktop companion built with **Python + PySide6**, powered by **DeepSeek API**.

Instead of being just another chatbot hidden inside a browser, she stays on your desktop, quietly accompanying your day.

---

## ✨ Features

### 🌸 Cute Chibi Companion

A tiny hand-drawn chibi girl lives on your desktop.

- Idle animation
- Walking animation
- Sleeping animation
- Speaking animation
- Drag & Drop interaction

She isn't just an icon.

She feels alive.

---

### 🤖 Powered by AI

Connected to the **DeepSeek API**, allowing natural conversations.

- Context-aware chatting
- Custom system prompts
- Adjustable personality
- Fast responses

She can answer questions...

or simply keep you company.

---

### 💬 Natural Interaction

Instead of opening a chat window,

just move your mouse near her.

A small input box appears beneath her.

Type your message.

Press **Enter**.

She'll reply in a speech bubble above her head.

Like talking to a tiny friend.

---

### 😴 Alive Behaviors

She doesn't stay still forever.

She will

- randomly walk around
- become sleepy after being idle
- sleep peacefully
- dream with floating **Z**
- wake up naturally
- occasionally start conversations herself

Sometimes...

she simply wants to say hello.

---

### ⚙️ Customizable

You can configure:

- DeepSeek API Key
- AI Model
- System Prompt
- Walking Speed
- Character Size
- Auto Talk
- Auto Walk
- Reply Duration

Everything can be adjusted to your preference.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourname/Cute-chibi-girl-deskpet.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python main.py
```

---

## 📦 Build

Generate a standalone executable with PyInstaller.

```bash
python -m PyInstaller desktop_pet.spec
```

The generated executable will be located in

```
dist/
```
